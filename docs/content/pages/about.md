Granit is built for personal use. That's the whole design philosophy: a focused note-taking app for one person's markdown cave, not a platform. No plugin system, no sync layer, no speculative abstractions — just the features that earn their place, kept deliberately small.

# Tech stack

| Layer | Tech |
|-------|------|
| Backend | Tauri 2 (Rust) |
| Frontend | Leptos 0.8 (Rust → WASM, CSR) |
| Styling | Tailwind CSS 4 + DaisyUI 5 |
| Markdown | `pulldown-cmark` |
| AI | `rig-core` |
| Errors | `thiserror` |
