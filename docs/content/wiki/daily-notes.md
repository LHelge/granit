Daily notes give you one note per day for journaling, logging, or planning. Granit can create or open today's note with a single action and seed it from a template, so each day starts from a consistent structure. Daily-note behavior is configured per cave — see [[configuration]] for the relevant settings.

# The daily-note folder

Each cave has a configurable daily-note folder. New daily notes are created there, keeping them grouped and separate from the rest of your notes. You set this folder in the cave's configuration; see [[configuration]] for details.

# Opening today's note

The daily-note button opens today's note if it already exists, or creates it if it does not. You never have to think about whether today's note exists yet — one action gets you to it either way.

# Template seeding

A new daily note can be seeded from a [[templates|template]]. Configure a daily-note template slug for the cave, and each freshly created daily note starts from that template's content — a heading layout, a task list, or whatever structure you want every day to begin with. Existing daily notes are never overwritten; seeding applies only when the note is first created.

> [!TIP]
> Pair a daily-note template with task lists so each day starts with a fresh checklist. Those tasks then flow into the Todo tab — see [[todos]].

# Calendar tab

The [[explorer]] includes a calendar that lets you browse daily notes by date. Use it to jump back to a previous day's note or to see which days already have notes.
